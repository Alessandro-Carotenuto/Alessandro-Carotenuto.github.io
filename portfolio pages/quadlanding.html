Overview

This project presents a complete autonomy pipeline for vision-based quadrotor landing on dynamic platforms. The system integrates high-frequency perception, state estimation, and cascaded PID control to enable autonomous docking on moving targets.

$$\hat{x}_{k} = \hat{x}_{k}^{-} + K_k(z_k - H\hat{x}_{k}^{-})$$

By fusing camera-based detection with a predictive Kalman Filter, the framework maintains robust tracking even during intermittent visual occlusions. The architecture was validated across complex trajectories (circular, figure-8) using a Simulink-CoppeliaSim co-simulation environment.

Key Elements

Visual Detection

Perspective-n-Point (PnP) geometry to recover the platform's 3D pose in real-time.

Flight Control

A dual-loop control architecture managing attitude stability and position tracking for landing maneuvers.

Predictive Planning

$p_{ref} = \hat{p} + \hat{v}\tau$

A feedforward trajectory generator that accounts for control lag by predicting the platform's future position

Kalman Filtering

$K_k = P_k^- H^T S_k^{-1}$

Robust state estimation and motion prediction
